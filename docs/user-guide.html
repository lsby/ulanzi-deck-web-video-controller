网页视频控制插件
Web Video Control User Guide

💡 核心解决痛点 & 开发初衷

在日常电脑使用中，我们经常会“一边看视频/小说，一边在其他软件里写代码或做其他工作”。

传统多媒体键痛点：键盘或系统自带的多媒体快捷键只能控制上一曲、下一曲和播放/暂停，无法控制进度（快进快退）。如果我们一时走神想要回退几秒重听，就必须中断手头工作，将窗口切换到浏览器，用鼠标拖动进度条，再切回工作窗口，非常打断专注状态。

本插件特色：在完全不切换当前窗口焦点、不中断手头工作的前提下，您可以通过 Ulanzi Deck 实体设备（按键或旋钮）直接微调后台浏览器的网页视频播放进度（快进/快退/暂停），帮助您始终保持专注。

1 🚀 第一步：配置浏览器与安装脚本

由于本插件需要控制浏览器网页内的视频播放，因此必须配合一个轻量级浏览器扩展使用。

  1. 运行注册表安装配置：

    请先在 Ulanzi 属性配置面板中点击 【安装配置】 按钮（或手动进入插件目录下的 scripts 文件夹并以管理员身份运行 install.cmd）。此脚本会自动写入 Native Messaging 策略白名单，允许浏览器扩展安全连接本地服务。

  2. 打开浏览器扩展程序管理页面：
    • Chrome 浏览器：访问 chrome://extensions/
    • Edge 浏览器：访问 edge://extensions/
  3. 开启开发者模式：

    在扩展程序管理页面中，找到并开启 “开发者模式” 开关。

    图：在管理页面中开启“开发者模式”
  4. 加载已解压的扩展程序：

    点击 “加载已解压的扩展程序 (Load unpacked)” 按钮，并在弹出的文件夹选择框中，选择插件根目录下的 chrome-plugin 文件夹即可。

    图：点击“加载已解压的扩展程序”选择 chrome-plugin 文件夹

2 🔗 第二步：确认连接状态

  1. 检查与调试连接：

    点击浏览器右上角工具栏的扩展图标，在弹窗中查看连接状态：

    • 若显示为“已连接”，说明通信链路已成功建立。
    • 若显示为“未连接”，可直接点击弹窗中的 【点击连接 Ulanzi】 按钮重新尝试连接。

3 🎛️ 第三步：配置动作与开始使用

  1. 打开 Ulanzi Studio 客户端软件。
  2. 在右侧的动作列表中，找到 网页控制 (Media Control) 类别。
  3. 将动作拖拽至您的实体 Deck 设备按键或旋钮上：
    • 视频快退：按下即可将视频进度后退 5 秒。
    • 视频快进：按下即可将视频进度前进 5 秒。
    • 播放/暂停：按下可自由切换视频的播放与暂停状态。
  4. 打开并手动播放视频：

    在浏览器中打开任意支持 HTML5 播放器的视频网站（如 Bilibili、YouTube 等）。

    💡 必须先手动点击播放一次

    进入视频页面后，您必须先在网页中手动点击播放一次该视频。这是因为浏览器需要通过“播放”动作来识别当前要控制哪一个视频目标。只要视频播放过一次，即便之后再次暂停，设备上的按键控制也将持续有效。

  5. 按下实体按键进行测试：

    现在按下您配置好的 Ulanzi Deck 实体按键或旋转旋钮，观察浏览器中的网页视频是否能够完美响应快进、快退及播放/暂停控制！

🔍 常见问题与排查

按下设备按键后，视频没有任何反应？
  • 排查一：请确保已经在配置面板中点击过 “安装配置” 并在弹出的命令行中执行成功。
  • 排查二：请确保已经在网页中手动点击播放过一次该视频。如果从未播放过，浏览器将不知道需要控制哪一个视频目标。只要视频播放过一次，即便在暂停状态下，按键操作也是有效的。
  • 排查三：点击浏览器右上角插件图标，查看连接指示是否正常。如果显示“未连接”，可尝试点击弹窗中的 【点击连接 Ulanzi】 按钮重新连接，或重启浏览器与 Ulanzi Studio 客户端。
